Cascade Assets

It is hard to manage assets inside of cascade.

Instead, organize styles, scripts, and images here, then upload the compressed, production-ready assets to cascade. The asset source files are located in the _assets/ folder. These assets get compiled into _sites/_assets/.

This project uses the Jekyll framework to serve a mini-local version of our cascade html templates. It also utilizes the jekyll-assets plugin to give us an asset pipeline similar to Ruby on Rails.

Ready to start developing!? Click on one of these sample templates, then start editing the source styles and scripts.
